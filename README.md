繁星组件库
====
基于vue2的移动端组件库，主要以交互组件为主，支持全局和按需加载。

``` bash
# Project setup
npm install

# dev
npm start

# 当代码中提示eslint报错时，可执行该命令进行修复
npm run fix

# Compiles and hot-reloads for development
npm run serve

# 组件库打包
npm run lib

# 文档开发
npm run docs:dev

# 文档打包
npm run docs:build

# 上传文档页面
npm run deploy
```
